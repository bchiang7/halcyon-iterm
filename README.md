# Halcyon for iTerm

![](images/demo.png)

## Installation

1. Clone this repo
1. In iTerm2 open `Preferences > Profiles`
1. Under the `Colors` tab find `Color Presets > Import`
1. Import the `halcyon.itermcolors` file
